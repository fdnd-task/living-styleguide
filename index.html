Typografie

Headers

Wel doen

Alle h1, h2, h3 moeten in CAPS zijn. Tekst in menu of navigatie moet monospace zijn.

Niet doen

Headers met lowercase; headers in italic.

Fonts

Wel doen

Helvetica Neue Light, Helvetica Neue, Helvetica, Arial, Roboto, fonts uit sans-serif categorie.
Voorbeeld van fonts die kunnen worden gebruikt:

Helvetica
Helvetica Neue Light, Helvetica Neue, Helvetica voor teksten en headers.
Roboto
Voor menu bar, buttons.

Niet doen

Er moeten geen fonts worden gebruikt met serif, geen calligraphic fonts.


  
  header {
    background-color: white;
    font-weight: 525;
    font-family: Helvetica;
    }

  .nav_bar {
    font-size: 14px;
    font-family: system-ui, -apple-system, 
    BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, 
    Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue',
    sans-serif;
    }

Structuur

Header

Wel doen

Naam van de website in ALL CAPS. Blijft boven aan de website staan.

Niet doen

Navigatie er inzetten. Als je scrolt gaat hij niet met je mee. 

  <header>
  <h1> Naam van de website </h1>
  </header>

  header {
    background-color: white;
    margin-bottom: 1rem;
    color: silver;
  }

  header > h1 {
    font-size: 1rem;
    text-align: right;
  }

Navigatie

Wel doen

Hamburger menu voor mobiel. Blijft gefixt links boven op het scherm. Op desktop is de nav bar gefixt boven aan de website. Dus het gaat mee als je aan het scrollen bent.

Niet doen

Niet transparant. Niet weining contrast.

Hamburger menu

  
  HTML
  <nav>
    <section class="mobileHeader laptopHeader">
      <div id="sideMenu" class="sideMenu">
          <a class="closebtn" onclick="closeMenu()">×</a>
          <a href="#"></a>
          <a href="#"></a>
        </div>
      <span class="menubtn" onclick="openMenu()">☰</span>
    </section>
  </nav>

  CSS
  .sideMenu {
    height: 100%;
    width: 0;
    position: fixed;
    z-index: 1;
    top: 0;
    left: 0;
    overflow-x: hidden;
    transition: 0.5s;
    padding-top: 20px;
  }

  .sideMenu a {
    padding: 0.5em 1em 0.5em 1em;
    text-decoration: none;
    font-size: 0.8em;
    display: block;
    transition: 0.3s;
  }

  .menubtn{
    font-size:30px;
    cursor:pointer;
    display:block;
    margin: 3% 5%;
  }

  JS
  function openMenu() {
    document.getElementById("sideMenu").style.width = "15vw";
  }

  function closeMenu() {
    document.getElementById("sideMenu").style.width = "0";
  }
