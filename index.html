Typografie

Wel doen

Headers

Alle h1, h2, h3 moeten in CAPS zijn.

Fonts

De fonts die moeten worden gebruikt zijn:

  • Arial
  • Sans-Serif

Niet doen

Vertel hoe dingen vooral niet toegepast moeten worden.

            <header>
            <h1>Zo maak je een header</h1>
            </header>

Structuur

Header

Wel doen

Naam van de website in ALL CAPS. Blijft boven aan de website staan.

Niet doen

Navigatie er inzetten. Als je scrolt gaat hij niet met je mee. 

<header>
  <h1> Naam van de website </h1>
</header>

header {
  background-color: white;
  margin-bottom: 1rem;
  color: silver;
}

header > h1 {
  font-size: 1rem;
  text-align: right;
}

Navigatie

Wel doen

Hamburger menu voor mobiel. Blijft gefixt links boven op het scherm. Op desktop is de nav bar gefixt boven aan de website. Dus het gaat mee als je aan het scrollen bent.

Niet doen

Niet transparant. Niet weining contrast.

JS
function openMenu() {
  document.getElementById("sideMenu").style.width = "15vw";
}

function closeMenu() {
  document.getElementById("sideMenu").style.width = "0";
}
